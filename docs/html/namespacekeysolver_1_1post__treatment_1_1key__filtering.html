def keysolver.post_treatment.key_filtering.filter_keys (   keys)

Turn a raw list of tagged frames into a list of key presses.

Parameters
keysA list representing, for each frame, the detected key as a string.
Returns
A list of key presses, in chronological order. Each key press is represented by an array of key strings, sorted from the most likely to the least likely. The "SHIFT" modifier is represented as a standalone key, just before the affected key press.
